# secret-img
encrypt/decrypt images

# Install
```console
$ npm i -g secret-img
```

# Usage
```console
$ secret-img <command>
```

# Commands
To encrypt images
```console
$ secret-img -e
```

To decrypt images
```console
$ secret-img -d
```

Check version
```console
$ secret-img -v
```

Help
```console
$ secret-img -h
```
